# Customer Purchase Behavior Analysis (Excel)

## Objective
Analyze ecommerce consumer behavior data to understand customer spending patterns across age groups and gender.

## Dataset
Ecommerce Consumer Behavior Analysis dataset sourced from Kaggle.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Sorting and Filtering
- Charts

## Analysis Performed
- Total purchase analysis by age group and gender
- Average spending behavior analysis
- Identification of high-revenue customer segments

## Key Insights
- Age groups 25–34 and 33–35 contribute the highest total revenue
- Female customers generate slightly higher total spend overall
- Younger age groups show higher average spend per transaction
